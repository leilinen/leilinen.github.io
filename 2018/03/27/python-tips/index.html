Python 中单引号，双引号，三引号的区别

面试的时候被问到这个问题，在平时使用中没有感觉单引号与双引号的区别，三引号主要用到多行的注释，后来又Google了一下。

我喜欢在用于插值的字符串或自然语言消息中使用双引号，并且对于小符号类型的字符串使用单引号，但是如果字符串包含引号或者我忘记了，则会打破规则。 即使不需要它们，我也可以为文档字符串和正则表达式的原始字符串文字使用三重双引号。

双引号还有一个区别，当字符串中有单引号的时候，需要使用使用双引号来加以区别。
