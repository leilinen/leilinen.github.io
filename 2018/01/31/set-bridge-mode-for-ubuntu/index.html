修改ubuntu16.04为桥接模式上网
2018-01-31| technology

分布式计算中需要用到分布式集群，而分布式集群需要为每一台服务器配置固定ip。vmware为虚拟机默认设置的上网方式为Nat模式，即虚拟机需要通过宿主主机网卡进行网络通信。虚拟系统每次开机，nat模式会自动分配一个IP给虚拟系统，因此虚拟系统每次开机取得的IP地址是不一样的，这样会给分布式集群造成很大的困扰，因此需要给虚拟主机配置桥接上网方式。

首先需要修改vmware的配置，将虚拟机的网络连接方式设置为桥接。

打开虚拟网络编辑器 **，选择更改设置**，选择相应网卡。

设置完毕后，启动虚拟机，设置虚拟机的IP地址，网关和子网掩码。
